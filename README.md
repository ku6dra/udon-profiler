# UdonProfiler
A Simple Profiler for VRCSDK3 Worlds.

## Requirements
### VRChat Client Version
- 2023.4.2p2 Build 1390 (2023-12-06)
### SDK
- Unity 2022.3.6f1
- VRChat SDK - Base v3.5.0
- VRChat SDK - Worlds v3.5.0

Compatibility with future updates is not guaranteed.
